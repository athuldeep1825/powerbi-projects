
Project Name: HR Analytics Dashboard

Tools Used:

Power BI
Excel

Description:

This project focuses on analyzing organizational employee data to derive actionable HR insights that support strategic workforce planning. The analysis evaluates hiring trends, workforce demographics, departmental distribution, and diversity metrics using interactive Power BI dashboards.

Key Insights:

Hiring trends show steady workforce growth after 2002 with a slight decline in recent years

Gender distribution is well balanced, indicating an inclusive workforce

Majority of employees are concentrated in Ohio, highlighting regional dependency

Engineering department has the highest workforce strength, while support functions are under-resourced

Workforce diversity analysis reveals opportunities to strengthen multicultural hiring initiatives

Outcome:

Provided data-driven recommendations to improve regional hiring balance, strengthen workforce diversity, optimize departmental staffing, and support informed HR decision-making through visual analytics.
