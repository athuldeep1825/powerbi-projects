# powerbi-projects
Data analytics dashboards created using Power BI

This repository contains a collection of Power BI data analytics projects focused on transforming raw datasets into actionable business insights. Each project demonstrates end-to-end analytical workflows including data cleaning, modeling, visualization, and insight generation across different business domains.

Key Highlights: 
Interactive Power BI dashboards and reports

Business-focused analytics (HR, Customer Segmentation, Sales, etc.)

Data preparation using Excel and Power Query

KPIs, trends, and insight-driven visual storytelling

Tools & Technologies:
Power BI

Excel

Power Query

DAX

Each folder represents an individual project with its own dataset, report, and documentation, showcasing practical data analytics and visualization skills.
